This project is an application written in Vue.js where most of the styling was done with it's vuetify library. It represents an online burger shop where you can order your favorite burger and it also has an admin panel where you can edit data. User data is placed at store so you can login with that credentials or you can register your account, but it only lasts one session.
Live version is available at:
https://burgerverse.vercel.app/
